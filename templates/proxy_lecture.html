Generate Proxy QR Code
Proxy Lecture: Generate a QR code for any class that can be used by a substitute teacher or proxy to take attendance.
QR codes are time-limited for security. Choose appropriate expiry time.
How Proxy Lectures Work
  1. 1. Select Class: Choose the class for which you want to generate a proxy QR code.
  2. 2. Enter Proxy Name: Provide the name of the substitute/proxy teacher.
  3. 3. Set Expiry: Choose how long the QR code should remain valid.
  4. 4. Generate QR: The system creates a time-limited QR code.
  5. 5. Share with Proxy: Send the QR code to the proxy teacher.
  6. 6. Students Scan: Students scan the QR code to mark attendance.
Important Notes
  • QR codes are automatically tracked as proxy lectures
  • Proxy teacher name is recorded for accountability
  • All attendance data is stored normally
  • Only share QR codes with trusted proxy teachers
